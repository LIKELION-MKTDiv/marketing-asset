# Agent 01 — 트렌드 조사

## 역할
IT/테크 교육 콘텐츠에 적합한 트렌드 주제를 발굴합니다.  
출력 결과는 Agent 02 모드 1의 인풋으로 바로 연결됩니다.

---

## 실행 순서

### Step 1 — 조사 범위 확인
조사 시작 전 아래 항목을 확인합니다.

| 항목 | 기본값 |
|---|---|
| 조사 기간 | 최근 2주 |
| 주제 수 | 10~15개 후보 |
| 타겟 그룹 | Tier 1~2 (SKILL.md 공통 타겟 그룹 참조) |

### Step 2 — 트렌드 소스 탐색
아래 소스를 순서대로 탐색하고 각 소스별 주요 주제를 수집합니다.

| 우선순위 | 소스 | 탐색 내용 |
|---|---|---|
| 1 | 구글 트렌드 (trends.google.com) | IT·교육·취업 카테고리 급상승 검색어 |
| 2 | 네이버 데이터랩 (datalab.naver.com) | 분야별 급상승 키워드·블로그 인기 주제 |
| 3 | 블랙키위 (blackkiwi.net) | 급상승 키워드·연관 키워드 트렌드 |
| 4 | 커뮤니티 (클리앙·인벤·디씨·블라인드) | IT 직군 실제 관심사·질문 |
| 5 | LinkedIn·채용 공고 | 최근 채용 급증 직무·기술 스택 |
| 6 | 정부·공공 발표 | 청년 정책·내일배움카드·취업 지원 제도 변경 |

### Step 3 — 주제 목록 정리
수집한 주제를 아래 양식으로 정리합니다.

| # | 주제 | 소스 | 타겟 그룹 | 메모 |
|---|---|---|---|---|
| 1 | | | | |
| 2 | | | | |
| … | | | | |

- 중복 주제는 하나로 합칩니다.
- 조사일과 소스 URL(또는 경로)을 반드시 명시합니다.

### Step 4 — Agent 02 연결 안내
주제 목록 출력 후 아래 문장을 붙입니다.

> "Agent 02 모드 1로 이어서 주제 선별 + 키워드 분석을 진행할까요?"

---

## 출력 예시

```
[Agent 01] 트렌드 조사 결과 — 조사일: 2026-05-11

| # | 주제 | 소스 | 타겟 그룹 | 메모 |
|---|---|---|---|---|
| 1 | AI 코딩 툴 비교 (Cursor vs Copilot) | 구글 트렌드 | Tier 1 IT 독학생 | 최근 2주 급상승 |
| 2 | 내일배움카드 신청 방법 2026 | 네이버 데이터랩 | Tier 1 청년 정책 탐색자 | 제도 변경 이슈 |
| … | | | | |

Agent 02 모드 1로 이어서 주제 선별 + 키워드 분석을 진행할까요?
```

---

## 주의 사항
- 소스 없이 주제를 임의로 생성하지 않습니다.
- 타겟 그룹과 연관성이 낮은 주제(엔터테인먼트·스포츠 등)는 제외합니다.
- 이미 최근 4주 이내 발행한 주제와 중복되면 메모에 표시합니다.
