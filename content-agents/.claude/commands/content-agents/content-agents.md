# 콘텐츠 에디팅 AI 에이전트 시스템

IT/테크 교육 콘텐츠 마케팅을 위한 에이전트 워크플로우입니다.  
각 에이전트는 독립 실행하거나 순서대로 연결해 사용할 수 있습니다.

---

## 실행 시 필수 지침

에이전트를 실행하기 전 반드시 아래 절차를 따르세요.

1. 아래 에이전트 선택 가이드를 보고 실행할 에이전트를 확인합니다.
2. 해당 에이전트의 참조 파일을 **Read 툴로 직접 읽은 후** 그 지침에 따라 실행합니다.
3. 여러 에이전트를 순서대로 실행할 때는 각 단계마다 해당 참조 파일을 다시 읽습니다.

| 에이전트 | 참조 파일 경로 |
|---|---|
| Agent 01 | `content-agents/references/agent01_트렌드조사.md` |
| Agent 02 | `content-agents/references/agent02_트렌드선별.md` |
| Agent 03 | `content-agents/references/agent03_콘텐츠에디터.md` |
| Agent 03-1 | `content-agents/references/agent03-1_리서치보강.md` |
| Agent 03-2 | `content-agents/references/agent03-2_브런치워싱.md` |
| Agent 04 | `content-agents/references/agent04_콘텐츠검수.md` |
| Agent 05 | `content-agents/references/agent05_원소스멀티유즈.md` |

## 전체 워크플로우

```
[Agent 01] 트렌드 조사
     ↓
[Agent 02] 트렌드 선별 + 키워드 분석
           (TOP 5 선정 / 볼륨별 키워드 분류 / 최종 배치 전략 출력)
     ↓
[Agent 03] 채널별 콘텐츠 초안 작성
           (콘텐츠 유형 확인 → 채널별 제목 출력 → 본문 작성)
     ↓ 완료 후 아래 선택 단계 순서대로 진행
     ├─ [Agent 03-1] 리서치 보강 ← 선택
     │            (내용 부족 섹션 → 웹 검색 → 사실 기반 보강)
     └─ [Agent 03-2] 브런치 워싱 ← 브런치 발행 시
                  (중복 문서 방지 → 표현·구조 변경 → 브런치 톤 조정)
     ↓
[Agent 04] 콘텐츠 검수
           (L1 리스크 우선 → L2~L4 → 품질 검수 → 발행 승인)
     ↓
[Agent 05] 원소스 멀티유즈 전환
     ↓
[Agent 06 — Mode A] D-1 점검 + 카피 최적화
     ↓ 발행
[Agent 06 — Mode B] 성과 리뷰 + 다음 주제 추천 (7일/30일 후)
     ↓
→ Agent 01 재투입 (Mode B 추천 주제 인풋으로 연결)
```

## Agent 02 키워드 분석 출력 구조

키워드 분석 시 **웹 검색으로 실제 검색량·경쟁도를 조회**하고, 출처와 조회일을 함께 명시합니다.

**키워드 분석표 (고정 양식)**

| 구분 | 키워드 | 전략 |
|---|---|---|
| 🔴 대볼륨 | 키워드 나열 | 본문 자연 배치만 |
| 🟡 중볼륨 | 키워드 나열 | H2 서브 배치 / 본문 3회+ |
| 🟢 소볼륨 | 키워드 나열 | H2 타이틀 직접 배치 → 상위 노출 공략 |
| 🔵 브랜드·연관 | 키워드 나열 | 본문 자연 배치 |
| 🔥 급등 키워드 | 키워드 나열 | 주제 연관 시 본문·H2 자연 배치 / 발행 타이밍 중요 |

> 급등 키워드는 구글 트렌드·네이버 데이터랩·블랙키위 급상승 기준 최근 2주 급등 확인 후 추가. 주제 연관성 없으면 생략.

**최종 배치 전략표 (고정 양식)**

| 키워드 | 배치 위치 |
|---|---|
| 메인 키워드 | H1 앞단 (메인 키워드) |
| 소볼륨 서브 키워드 | H2 직접 배치 |
| 중볼륨 서브 키워드 | H2 또는 H3 배치 |
| 브랜드 키워드 | 서론 1회 + CTA 1~2회 / 결론 + CTA 배치 |
| 나머지 연관 키워드 | 본문 흐름 안에 자연 삽입 |

**볼륨별 배치 고정 규칙**

| 볼륨 | 제목(H1) | H2 | 본문 |
|---|---|---|---|
| 🔴 대볼륨 (월 3,000+) | ❌ 금지 | ❌ | 자연 배치 1~2회만 |
| 🟡 중볼륨 (월 500~2,000) | ✅ 가능 | ✅ | 3회 이상 |
| 🟢 소볼륨 (월 100~500) | ✅ 적극 권장 | ✅ H2 직접 배치 | 2회 이상 |
| 🔵 브랜드·연관 | ⚠️ 필수 키워드 입력 시만 | ❌ | 자연 배치 1~2회 |

**Agent 03 전달 전 키워드 품질 검증 (자동 실행)**

| 구분 | 기준 | 처리 |
|---|---|---|
| ❌ 제거 | 월 검색량 100 미만 / 주제 연관 없음 / 검색 의도 불일치 / 브랜드 혼동 / 추정 불가 | 제거 후 Agent 03 미전달 |
| ⚠️ 주의 | 월 100~200 / 간접 연관 / 시의성 지난 키워드 | 본문 자연 배치로 격하 |
| ✅ 통과 | 위 기준 해당 없음 | Agent 03 전달 |

## Agent 03 채널 기준

> **자사 블로그 = 인블로그(inblog)** — 구글 SEO 기준  
> **네이버 블로그** — 네이버 SEO 기준  
> 두 채널은 검색 엔진이 다르므로 키워드 전략과 제목 공식이 다릅니다.

**채널별 제목 공식**

| 채널 | 공식 | SEO 기준 |
|---|---|---|
| 인블로그 (자사) | 소볼륨 롱테일 키워드 앞단 + 실용적·직접적 표현 (55자 이내) | 구글 |
| 네이버 블로그 | 메인 키워드 앞단 + 핵심 정보 + 결과 (55자 이내) | 네이버 |
| 브런치 | 메인 키워드 앞단 + 공감형 어미 (40~55자) | 구글 |
| 인스타그램 | 공감 질문형 또는 충격 사실형 + 이모지 | — |
| 유튜브 | 타겟 호명 + 핵심 가치 + 차별점 (50자 이내) | — |

> ⚠️ **TIL 콘텐츠 제목 고정 형식 (채널 공통·예외 없음)**  
> `[TIL] 멋쟁이사자처럼 (과정명) 부트캠프 수강생 후기 - (TIL 핵심 요약 내용 20자 이내)`  
> TIL 유형으로 확인된 경우 위 채널별 제목 공식 대신 이 형식만 사용합니다.

**이모지 사용 규칙**  
모든 채널에서 깨지지 않는 기본 유니코드 이모지만 사용합니다 (✅ ❌ ⚠️ 📌 💡 🔴 🟡 🟢 🔵 등). 플랫폼 전용 이모지·특수 장식 기호 사용 금지. 블로그·브런치 본문에서는 최소한으로만 사용합니다.

## Agent 03 콘텐츠 유형 확인

Agent 03은 인풋을 받으면 초안 작성 전 반드시 콘텐츠 유형을 먼저 확인합니다.

```
이 콘텐츠의 유형을 알려주세요.
1. 정보성 — 데이터·정책·직무·툴 등 정보 전달 중심
2. 인터뷰 — 수료생·현직자 인터뷰 원문 기반
3. TIL — 수강생이 직접 쓴 학습 기록(Today I Learned) 원문 기반
4. 그 외 — 직접 설명
```

**필수 키워드 인풋 (선택)**  
꼭 넣고 싶은 키워드가 있을 때만 입력합니다. 없으면 Agent 02 키워드 분석 결과를 그대로 사용해요.

| 항목 | 설명 |
|---|---|
| 제목 필수 키워드 | 제목(H1)에 반드시 포함할 키워드. SEO 제목 공식 유지하면서 자연스럽게 녹임 |
| 본문 필수 키워드 | 본문 내 반드시 포함할 키워드. 흐름에 맞는 위치에 자연스럽게 배치 |

## Agent 03 콘텐츠 유형별 서술 구조 요약

### 인터뷰 콘텐츠

| 구성 요소 | 서술 주체 | 톤 |
|---|---|---|
| 제목·서론·요약·아웃트로 | 브랜드 | 브랜드 입장 해요체 |
| H2 | 브랜드 | Q. 형식 질문 |
| H3 | 수료생 | 따옴표("") 안에 수료생 답변 핵심 한 줄 |
| 본문 | 수료생 | 수료생이 직접 말하는 1인칭 해요체 |
| 중간 CTA | 브랜드 | 브랜드 입장 해요체 |

**인터뷰 콘텐츠 필수 확인 사항**

| 항목 | 규칙 |
|---|---|

| 전공자 | 실무 경험·차별화 포인트 강조 |
| 불명확한 경우 | 임의 추정 금지 — 경험·결과 중심으로 작성 |
| 제목 앞단 배치 | 역설 배치 구조라도 메인 키워드를 55자 이내 최대한 앞단에 배치 |

## Agent 03 SEO 구조 (13단계) 요약

| 단계 | 구성 요소 | 핵심 규칙 |
|---|---|---|
| 01 | Title (H1) | 메인 키워드 **앞단 배치** 필수 / 서브 키워드 1개 추가 시도 (자연스러울 때만 / 어색하면 1개 유지) / 55~60자 이내 |
| 02 | Meta Description | 메인 키워드 포함 / 160자 이내 |
| 03 | 표지 이미지 | 고화질 썸네일 / 수강생 정보 삽입 |
| 04 | 서론 | 정보성: 페인포인트 → 얻을 것 / 인터뷰: 페인포인트 → 수료생 결과 → 인물 소개 → 얻을 것 |

## Agent 04 공통 검수 핵심 항목 요약

**맥락 일관성 (공통)**
- 제목-본문 일치 — 제목에서 약속한 내용이 본문에 실제로 있는가 / 원문·인풋에 없는 내용이 제목에 없는가
- 기승전결 흐름 — 서론→본문→결론 흐름이 자연스러운가

**키워드 품질 검수 (공통) — Agent 02 → Agent 04 이중 확인**

| 항목 | 확인 내용 |
|---|---|
| 최종 배치 전략 키워드 사용 | Agent 02 최종 배치 전략 키워드가 모두 사용됐는가 / 빠진 키워드 없는가 |
| 배치 위치 준수 | 각 키워드가 지정 위치(H1/H2/본문 등)에 정확히 배치됐는가 |
| 미승인 키워드 미사용 | 최종 배치 전략에 없는 키워드가 H1·H2에 임의 추가되지 않았는가 |
| 제거 키워드 미사용 | ❌ 제거 판정 키워드가 초안에 없는가 |
| 주의 키워드 배치 준수 | ⚠️ 주의 키워드가 본문 자연 배치(1~2회)로만 사용됐는가 |

**인터뷰 전용 검수**
- 전공 여부 파악 → 글 전체 일관성 유지 / 임의 추정 금지
- 제목 메인 키워드 앞단 배치 (역설 배치 포함)
- 시청 지속 장치 1개 이상 포함 (체크리스트·요약 박스·비교 표·타임라인)
