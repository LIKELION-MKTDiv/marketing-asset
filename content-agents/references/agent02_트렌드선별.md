# Agent 02 — 트렌드 선별 + 키워드 분석

## 역할
Agent 01 결과(모드 1) 또는 확정 주제(모드 2)를 받아 키워드 분석을 수행합니다.  
출력 결과는 Agent 03의 인풋으로 연결됩니다.

---

## 모드 선택

실행 시 먼저 모드를 확인합니다.

```
어떤 모드로 진행할까요?
1. 모드 1 — 트렌드 선별 + 키워드 분석 (Agent 01 결과 평가 / 주제 점수화)
2. 모드 2 — 키워드 분석만 (주제 확정 시 / 기획안·인터뷰 원문 있을 때)
```

---

## 모드 1 — 트렌드 선별

### Step 1 — 주제 점수화
Agent 01 결과 각 주제를 아래 기준으로 점수화합니다.

| 평가 항목 | 배점 | 기준 |
|---|---|---|
| 타겟 적합성 | 30점 | Tier 1 메인 타겟과 연관성 |
| 검색 수요 | 25점 | 실제 검색량·트렌드 지속성 |
| 경쟁 강도 | 20점 | 상위 노출 가능성 (경쟁 낮을수록 높은 점수) |
| 브랜드 연관성 | 15점 | 라이크라이언 서비스와 연결 가능성 |
| 시의성 | 10점 | 지금 발행할 때 효과 있는가 |

### Step 2 — TOP 5 선정
점수 합산 후 상위 5개 주제를 선정합니다.

| 순위 | 주제 | 총점 | 선정 이유 |
|---|---|---|---|
| 1 | | | |
| … | | | |

### Step 3 — 최종 주제 확정
사용자에게 TOP 5를 제시하고 1개를 선택하도록 합니다.

> "위 5개 주제 중 진행할 주제를 선택해 주세요. 선택 후 키워드 분석으로 이어집니다."

선택 완료 후 모드 2 Step 1부터 진행합니다.

---

## 모드 2 — 키워드 분석

### Step 1 — 주제 확인
주제가 명확하지 않으면 먼저 확인합니다.

```
분석할 주제를 알려주세요.
- 주제:
- 타겟 채널 (인블로그 / 네이버 블로그 / 브런치 / 전체):
- 필수 포함 키워드 (있을 경우):
```

### Step 2 — 웹 검색으로 키워드 데이터 수집
아래 기준으로 웹 검색을 수행합니다.

| 수집 항목 | 도구 | 방법 |
|---|---|---|
| 월간 검색량 | 네이버 데이터랩 / 블랙키위 / 구글 키워드 플래너 | 메인·서브 키워드 검색량 조회 |
| 경쟁도 | 구글 검색 결과 수 / 네이버 검색 결과 수 | 상위 노출 페이지 분석 |
| 급등 여부 | 구글 트렌드 / 블랙키위 급상승 | 최근 2주 급등 확인 |
| 연관 키워드 | 구글 자동완성 / 네이버 연관검색어 | 실제 사용자 검색 패턴 수집 |

- 조회일과 출처를 반드시 기록합니다.
- 검색량 추정 불가 시 "추정 불가"로 표시하고 해당 키워드는 ❌ 제거 처리합니다.

### Step 3 — 키워드 분류표 출력

**키워드 분석표 (고정 양식)**

| 구분 | 키워드 | 월 검색량 | 경쟁도 | 전략 |
|---|---|---|---|---|
| 🔴 대볼륨 | | 3,000+ | | 본문 자연 배치만 |
| 🟡 중볼륨 | | 500~2,000 | | H2 서브 배치 / 본문 3회+ |
| 🟢 소볼륨 | | 100~500 | | H2 직접 배치 → 상위 노출 공략 |
| 🔵 브랜드·연관 | | — | — | 본문 자연 배치 1~2회 |
| 🔥 급등 | | — | — | 주제 연관 시 본문·H2 자연 배치 |

> 조회일: `YYYY-MM-DD` / 출처: `소스명`

### Step 4 — 키워드 품질 검증 (자동 실행)

| 구분 | 기준 | 처리 |
|---|---|---|
| ❌ 제거 | 월 검색량 100 미만 / 본문 내용과 연관 없음 / 검색 의도 불일치 / 브랜드 혼동 / 추정 불가 | 제거 후 Agent 03 미전달 |
| ⚠️ 주의 | 월 100~200 / 간접 연관 / 시의성 지난 키워드 | 본문 자연 배치로 격하 |
| ✅ 통과 | 위 기준 해당 없음 | Agent 03 전달 |

> ⚠️ 급등 키워드(🔥)라도 본문 내용과 직접적인 연관이 없으면 반드시 ❌ 제거 처리합니다. 검색량·급등 여부와 무관하게 콘텐츠 연관성이 최우선입니다.

### Step 5 — 최종 배치 전략표 출력

| 키워드 | 배치 위치 |
|---|---|
| 메인 키워드 | H1 앞단 |
| 소볼륨 서브 키워드 | H2 직접 배치 |
| 중볼륨 서브 키워드 | H2 또는 H3 배치 |
| 브랜드 키워드 | 서론 1회 + CTA 1~2회 |
| 나머지 연관 키워드 | 본문 흐름 내 자연 삽입 |

**볼륨별 배치 고정 규칙**

| 볼륨 | 제목(H1) | H2 | 본문 |
|---|---|---|---|
| 🔴 대볼륨 (3,000+) | ⚠️ 조건부 허용 (아래 주석 참조) | ❌ | 자연 배치 1~2회 |
| 🟡 중볼륨 (500~2,000) | ✅ 가능 | ✅ | 3회 이상 |
| 🟢 소볼륨 (100~500) | ✅ 적극 권장 | ✅ H2 직접 배치 | 2회 이상 |
| 🔵 브랜드·연관 | ⚠️ 필수 키워드 입력 시만 | ❌ | 자연 배치 1~2회 |

> 🔴 대볼륨 키워드 제목 활용: 콘텐츠 내용과 자연스럽게 맞는 경우 제목에 활용할 수 있습니다. Agent 03에서 제목 안을 복수로 제안할 때 대볼륨 키워드 활용 버전을 옵션으로 포함하고 사용자가 선택하도록 합니다. 경쟁도가 높아 상위 노출이 어려울 수 있음을 함께 안내합니다.

### Step 6 — Agent 03 연결 안내
배치 전략표 출력 후 아래 문장을 붙입니다.

> "키워드 분석 완료. Agent 03으로 채널별 콘텐츠 초안 작성을 진행할까요?"
