# KDT 모객현황 데일리 리포트

KDT 부트캠프 **지원서 CSV → 인터랙티브 HTML 대시보드 + Obsidian용 MD 노트**를 한 번에 생성하는 파이프라인.
모객 현황을 캠프별로 진단하고, 전일 대비 추세·마감일 도달 전망까지 자동으로 인사이트를 뽑아줍니다.

> 소스 로직: `kdt 모객현황 데이터 추출.ipynb` 기반 (내부직원·테스트 지원 제외 처리 포함)

## ⚙️ 설정 (부트캠프 추가·목표 변경)

`generate_daily_report.py` 상단 4개 dict만 수정하면 됩니다. **키는 표시명(DISPLAY 값) 기준.**

```python
# 파일 prefix → 보기 좋은 표시명  (prefix = CSV 파일명에서 '_' 앞부분)
DISPLAY = {
    "kdt-aiplus":      "AI+ NLP 5기",
    "kdt-backendj-26th":"백엔드 자바 26기",
    "kdt-growth-6th":  "그로스 마케팅 6기",
}

TARGETS  = { "AI+ NLP 5기": 96,  "백엔드 자바 26기": 72,  "그로스 마케팅 6기": 142 }  # 목표 모객수
CAPACITY = { "AI+ NLP 5기": 40,  "백엔드 자바 26기": 24,  "그로스 마케팅 6기": 57  }  # 수강정원
PERIODS  = {  # 모객 기간 (시작, 마감)
    "AI+ NLP 5기":     ("2026-05-18", "2026-06-22"),
    "백엔드 자바 26기": ("2026-05-27", "2026-06-29"),
    "그로스 마케팅 6기": ("2026-06-16", "2026-07-19"),
}
```

- 새 부트캠프는 CSV만 폴더에 넣고 위 dict에 4줄 추가하면 자동 반영
- 목표/정원/기간 **미설정 시** 해당 지표는 `미설정`·`—`로 표시 (에러 없음)

## 📊 리포트가 담는 인사이트

### Summary 탭
- 전체 요약 KPI (총 지원시작·완료·미완료 적체·내배카 보유율)
- **모집 마감 전망 (트렌드 × 종결일)** — 캠프별 마감 D-day, 현재 페이스로 마감일 예상 도달치, 목표 달성 전망(조기달성/달성전망/근접·미달위험/미달위험)
- **전일 대비 경향성 (비판적 분석)** — 직전 리포트 자동 로드해 핵심 지표 Δ(증감)와 경향성 코멘트. 목표 변경 시 직전 달성율을 현재 목표로 재계산해 왜곡 방지
- 캠프별 퍼널 비교, 만나이 분포 박스플롯(캠프 비교)

### 부트캠프별 탭
- **현황 점검** — 목표 모객수/수강정원/완료/목표달성율/경쟁률 + 목표 대비 상태 진단
- **모집 마감 전망 박스** — 페이스 기반 마감일 도달 시뮬레이션 진단
- **점검 코멘트 (비판적)** — 완료전환 병목, 비용민감 세그먼트, 마감 페이스 대비 여유/빠듯/불가 등 데이터 신호 기반 2~3개 자동 도출
- **누적 지원시작 vs 완료 + 예상 시뮬레이션(점선)** — 일자별 수치 라벨 + 마감일까지 페이스 외삽 점선
- 일별 신규 지원, 연령대 분포, **일일 만나이 박스플롯**(Q1·중앙·Q3 라벨)
- **연령대 × 합불상태** 피벗 (현재 심사 단계 분포)
- **내일배움카드 교차분석** (연령·성별·신분 × 내배카, melt·cast)

## 🧭 그로스 마케팅팀 활용 팁

1. **매일 아침 1회 실행** → `reports/` 또는 Obsidian에서 당일 리포트 확인
2. **마감 전망**부터 보기 — "예상 도달 / 목표 대비 %"로 가속이 필요한 캠프를 즉시 식별
3. **전일 대비 경향성**의 ▼(하락) 신호 주목 — 유입은 느는데 전환율이 떨어지면 채널 품질·지원서 이탈 점검
4. **경쟁률(지원/정원)**과 **내배카 보유율**로 모객 질·세그먼트 확인
5. 캠프가 추가되면 CSV 넣고 설정 dict 4줄만 추가 → 별도 코드 수정 불필요

> ⚠️ 현재 TEST 출력본 — 지표 정의는 팀 협의 후 정식화 예정. `모집 7일 미만` 캠프의 마감 전망은 `⚠️초기`로 표시되며 추정 신뢰도가 낮으니 며칠 관찰 후 판단하세요.
