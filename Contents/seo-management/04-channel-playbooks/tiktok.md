# 채널 플레이북 — 틱톡

> 인스타그램과 마찬가지로 검색엔진에 직접 색인되지 않는 폐쇄형 플랫폼(단, 틱톡 자체 검색 사용률이 높아 "플랫폼 내 SEO"의 비중이 인스타그램보다 크다).

---

## 1. 플랫폼 내 검색 최적화

| 항목 | 규칙 |
|---|---|
| 캡션 키워드 | 틱톡은 자체 검색 엔진 역할이 커지고 있음 — 캡션 첫 문장에 핵심 키워드 자연 포함 필수 |
| 음성/자막 텍스트 | 영상 내 음성과 화면 자막도 틱톡이 색인에 활용 → 자막에도 키워드 포함 |
| 해시태그 | 콘텐츠 관련 3~5개 핵심 해시태그(인스타그램보다 적게, 관련성 높게) |
| 사운드/트렌드 | 트렌드 사운드 사용이 탐색 노출에 영향 — 다만 브랜드 메시지가 묻히지 않게 |
| 계정 바이오 | 검색 가능한 키워드 포함 |

## 2. 콘텐츠 원칙

| 항목 | 규칙 |
|---|---|
| 도입부 1~2초 | 후킹 문구/비주얼로 이탈 방지(완주율이 핵심 랭킹 신호) |
| 완주율 최적화 | 15~30초 내외로 핵심 메시지 압축 |
| 텍스트 오버레이 | 무음 시청자를 위해 핵심 메시지를 화면 텍스트로도 노출 |
| 정보형 콘텐츠 | 체크리스트/비교형 콘텐츠는 저장·공유 유도에 유리 |

## 3. 정본 연계

유튜브 숏폼이나 블로그 콘텐츠의 파생본으로 운영 시 `../02-content-seo/cross-publishing.md` 원칙에 따라 프로필 링크에 정본 연결. 자동 자막을 그대로 복붙하지 말고 틱톡 톤에 맞게 재작성.
