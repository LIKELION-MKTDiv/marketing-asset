# 채널 플레이북 — 마이크로사이트 / 이벤트 페이지

> 캠페인·이벤트·특정 기수 모집 등 **단기 운영되는 독립 페이지**. 홈페이지 서브도메인 또는 별도 도메인으로 운영되는 경우가 많다.

---

## 1. 마이크로사이트 특유의 SEO 리스크

| 리스크 | 대응 |
|---|---|
| 별도 도메인/서브도메인 사용 시 도메인 권위가 분산됨 | 가능하면 본 도메인의 서브디렉토리(`example.com/campaign/`)로 운영 |
| 이벤트 종료 후 방치되는 죽은 페이지(dead page)가 쌓임 | 종료 시 301 리다이렉트로 관련 상시 페이지에 연결. 방치 금지 |
| 짧은 운영 기간이라 콘텐츠 SEO 투자를 생략하기 쉬움 | 최소한의 title/description/H1은 반드시 적용 — 짧은 기간에도 검색 유입은 발생 |
| 중복 콘텐츠(본 사이트 상세페이지와 내용 겹침) | `noindex` 처리하거나 캐노니컬을 본 페이지로 지정 |

---

## 2. 테크니컬 SEO 최소 기준

마이크로사이트는 `../01-technical-seo/checklist.md`의 "만들고 끝"이 아니라 종료 시점까지 고려한 라이프사이클 관리가 필요하다.

| 단계 | 할 일 |
|---|---|
| 오픈 전 | title/description/H1 설정, 대표 이미지(OG) 설정, HTTPS 확인 |
| 운영 중 | 서치콘솔에 URL 등록(색인 요청), 성과 모니터링 |
| 종료 시 | 301 리다이렉트 설계(관련 상시 페이지로) 또는 `noindex` + 안내 문구, 사이트맵에서 제거 |

---

## 3. 콘텐츠 SEO는 축약형으로

이벤트 페이지는 정보량보다 전환이 우선이므로 `../02-content-seo/writing-rules.md`의 "정보 탐색형" 대신 "탐색 유저 대상(전환 유도형)" 구조를 기본으로 쓴다. 상세 정보는 본 사이트 상세페이지로 링크.

## 4. GEO/AEO

단기 페이지는 AI 답변 인용 가치가 낮으므로 우선순위 낮음. 상시 운영되는 랜딩(구독형 서비스 소개 등)이라면 `../03-geo-aeo/geo-aeo-guide.md` 적용 검토.
