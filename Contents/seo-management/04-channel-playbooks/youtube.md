# 채널 플레이북 — 유튜브

> 유튜브는 자체 추천 알고리즘 + **구글 검색·AI 개요에도 함께 색인**되는 하이브리드 채널. 세 층위(테크니컬/콘텐츠/GEO-AEO) 원칙이 모두 부분 적용된다.

---

## 1. 테크니컬 요소 (유튜브 자체 + 구글 색인)

| 항목 | 규칙 |
|---|---|
| 제목 | 60자 이내, 핵심 키워드 앞배치(구글 검색 결과 타이틀로도 노출) |
| 설명란 | 상단 2~3줄에 핵심 요약 + 키워드(검색 결과 스니펫으로 노출). 하단에 정본 블로그/홈페이지 링크(UTM) |
| 챕터(타임스탬프) | `0:00 인트로` 형식으로 구간 표시 → 구글 검색에 챕터별 딥링크로 노출 가능 |
| 자막(캡션) | 정확한 자막 파일 업로드(자동 생성 자막 방치 금지) — 검색엔진이 자막 텍스트를 색인 |
| 썸네일 | alt 텍스트 개념은 없지만 CTR에 직접 영향 → 텍스트 오버레이로 핵심 메시지 전달 |
| 파일명 | 업로드 전 파일명에 키워드 포함(마이너하지만 참고 신호) |
| 재생목록 | 주제별로 묶어 채널 전체 주제 일관성 강화(탐색 알고리즘 + 구글 색인 모두에 긍정적) |

## 2. 콘텐츠 요소

| 항목 | 규칙 |
|---|---|
| 태그 | 영상 주제 관련 키워드 5~10개(과도한 인기 태그 남용 금지) |
| 카드/최종화면 | 관련 영상·구독 유도로 시청 지속시간 확보(알고리즘 핵심 지표) |
| 시청 지속시간 최적화 | 도입부 15초 내 핵심 메시지 전달(이탈 방지) |
| 댓글 고정 | 핵심 CTA 또는 요약을 고정 댓글로 배치 |

## 3. GEO/AEO 연계

구글 AI 개요·AI 모드는 관련성 높은 영상을 답변에 함께 노출한다. 설명란의 명확한 요약 + 정확한 자막이 인용 가능성을 높인다(`../03-geo-aeo/geo-aeo-guide.md` §3-2 "명확한 답변 구조" 원칙과 동일하게 적용).

## 4. 정본 연계

블로그 콘텐츠를 원안으로 한 영상이면 설명란에 정본 링크 필수(`../02-content-seo/cross-publishing.md`). 반대로 영상이 원안이고 블로그로 파생시키는 경우, 자막을 그대로 붙여넣지 말고 글 형식에 맞게 재구성.

## 5. 발행 전 체크리스트
- [ ] 제목 60자 이내 + 키워드 앞배치
- [ ] 설명란 상단 요약 + 정본 링크(UTM)
- [ ] 정확한 자막 업로드
- [ ] 챕터 타임스탬프 설정(4분 이상 영상)
- [ ] 관련 재생목록에 포함
- [ ] 썸네일 텍스트 가독성 확인
