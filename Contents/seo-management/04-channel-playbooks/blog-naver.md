# 채널 플레이북 — 네이버 블로그

> 공통 원칙(`../02-content-seo/writing-rules.md`)에 얹는 네이버 블로그 전용 규칙. 출처: `content-project/SEOREADME.md` §3-1.

| 항목 | 특성 및 규칙 |
|---|---|
| 주요 검색엔진 | 네이버 (VIEW 탭 노출이 핵심) |
| 핵심 알고리즘 | C-Rank(주제 전문성·일관성) + D.I.A(사용자 반응 기반 정보 품질) |
| 제목 | 핵심 키워드 앞배치. 80~100자 이내(간결 권장) |
| 본문 분량 | 최소 1,500자, 권장 2,000~2,500자 |
| 키워드 밀도 | 메인 키워드 본문 내 3~5회 자연 삽입, 제목에 1회 |
| 소제목 | 네이버 에디터 '소제목' 서식 사용 → H2/H3으로 파싱됨 |
| 이미지 | 텍스트 우선. 정보를 이미지로만 담지 말 것(검색봇은 텍스트 우선 파싱) |
| 태그 | 게시 시 관련 태그 5~10개(메인/연관 키워드 포함) |
| 채널 최적화 | 1주제 집중 운영(C-Rank 주제 일관성), 정기 발행(활성화 채널 우대) |
| 상호작용 | 댓글 관리, 공감/이웃 활동이 VIEW 노출에 영향 |
| 테크니컬 SEO | 플랫폼이 대부분 자동 처리 — 직접 통제 불가 |
| GEO/AEO | 낮음 — 네이버 자체 알고리즘이라 구글 AI 개요 인용 가능성 낮음 |
| 주의사항 | 인기 키워드 억지 삽입 → 스팸 분류 위험. 대가성 포스트 미표기 → 패널티 |

## 발행 전 체크리스트
- [ ] 네이버 키워드 도구에서 검색량/경쟁도 확인
- [ ] 검색 결과 상위 5개 콘텐츠의 구조/분량/키워드 벤치마킹
- [ ] 네이버 서치어드바이저에 블로그 등록 + 사이트맵/RSS 제출
- [ ] 이미지 ALT 텍스트 설정
- [ ] 중복 콘텐츠(다른 플랫폼과 동일 본문) 회피 — `../02-content-seo/cross-publishing.md`
