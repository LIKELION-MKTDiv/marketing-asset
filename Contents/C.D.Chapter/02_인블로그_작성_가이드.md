# 인블로그 작성 가이드

> 보이스 & 톤은 [01_브랜드_가이드.md](./01_브랜드_가이드.md)를 따릅니다.

---

## 글 유형

| 유형 | 목적 | 예시 주제 |
|---|---|---|
| **정보성** | SEO + 브랜드 전문성 구축 | "코딩 교육 시작 전 알아야 할 것 5가지" |
| **스토리** | 감성 전환 + 신뢰 구축 | "수강생 합격 후기: 6개월 만에 취업" |
| **홍보성** | 직접 전환 | "2분기 코스 모집 안내" |

---

## 기본 글 구조

```
1. 제목 (H1)       — 검색 키워드 + 궁금증 유발, 60자 이내
2. 표지 이미지      — 수강생 사진 or 브랜드 썸네일
3. Meta Description — 3줄 요약 (SERP 노출 기준)
4. 핵심 요약 + 목차 — 상단 배치, 신뢰 지표 역할
5. 본문 (H2/H3)    — 질문-답변 구조, 키워드 자연 배치
6. 이미지           — 텍스트 5~7줄마다 1장, alt 텍스트 필수
7. CTA              — 내부 링크 + 버튼형 배너
```

---

## SEO 템플릿 (단계별)

### 01. 제목 (H1)
- 규칙: `[카테고리] + 숫자 성과 + 핵심 키워드` (60자 이내)
- 핵심 키워드를 앞쪽에 배치해 CTR 극대화
- 예: `[그로스 마케팅] 비전공자 6개월 만에 취업 성공한 3가지 이유`

### 02. 표지 이미지
- 수강생 프로필 사진 또는 브랜드 썸네일
- 수강생 정보(전공/부트캠프/이름) 포함

### 03. Meta Description (서론)
- 검색 결과창(SERP) 노출 기준으로 핵심 키워드 포함
- 네이버는 서론 첫 문장으로 대체

### 04. 핵심 요약 + 목차
- `[누가 / 무엇을 / 결과]` 3줄 요약 박스
- 클릭 이동형 목차 상단 배치

### 05 & 06. 본문 (H2 & H3 위계)
- **H2 (주요 질문)**: 답변을 유추할 수 있는 질문형 키워드
- **H3 (상세 답변)**: 답변 핵심을 담은 한 줄 요약

```
예시)
H2: "비전공자가 IT 직무로 커리어 전환을 결심한 결정적 계기는?"
H3: "실무 역량 부재를 해결하기 위한 체계적인 교육이 절실했습니다."
```

### 07. 본문 작성 원칙
- 단문 위주, 핵심 먼저
- 키워드 자연 배치 (억지 반복 금지)
- 최소 800자 이상

### 08. 이미지
- 텍스트 5~7줄마다 이미지 1장 (평균 4~5개)
- alt 텍스트 반드시 입력

### 09. CTA
- 관련 콘텐츠 내부 링크 (체류 시간 증가)
- 버튼형 배너로 전환 유도
