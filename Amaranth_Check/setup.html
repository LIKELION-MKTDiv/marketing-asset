구글 시트 복사

출퇴근 기록용 템플릿 시트를 내 계정에 복사합니다

  1. 아래 버튼을 클릭하면 구글 시트 복사 화면이 열립니다.
    📋 구글 시트 복사하기
  2. 구글 계정 로그인 후 "사본 만들기" 버튼을 클릭합니다
  3. 시트가 열리면 주소창의 URL에서 Spreadsheet ID를 복사합니다
    URL 형식: https://docs.google.com/spreadsheets/d/[여기가 ID]/edit
    예시: 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms
  4. 아래 입력칸에 복사한 ID를 붙여넣으세요
URL에서 /d/ 와 /edit 사이의 문자열입니다
Spreadsheet ID를 입력해주세요

GCP 프로젝트 생성

구글 API를 사용하려면 GCP에 앱을 등록해야 합니다

  1. 아래 버튼으로 GCP 콘솔을 엽니다 (같은 구글 계정으로 로그인)
    GCP 콘솔 열기 →
  2. 화면 상단 구글 로고 오른쪽 "프로젝트 선택" 드롭다운 클릭
  3. 팝업 우측 상단 "새 프로젝트" 클릭
  4. 프로젝트 이름 입력 (예: amaranth-check) → "만들기" 클릭
  5. 잠시 후 상단에 만든 프로젝트 이름이 표시되면 완료
다음 단계부터는 방금 만든 프로젝트가 선택된 상태여야 합니다.
상단에 프로젝트 이름이 표시되는지 확인하세요.

Sheets API 활성화 & OAuth 동의 화면

구글 시트 접근 권한과 앱 등록을 완료합니다

① Google Sheets API 활성화

  1. Sheets API 설정 열기 →
  2. 파란색 "사용" 버튼 클릭
    ("API 사용 중지"로 표시되면 이미 활성화된 것 — 다음으로)

② OAuth 동의 화면 설정

  1. OAuth 동의 화면 열기 →
  2. User Type: "외부" 선택 → "만들기"
  3. 앱 정보 입력:
    · 앱 이름: 아무거나 (예: Amaranth Check)
    · 사용자 지원 이메일: 드롭다운에서 내 이메일 선택
    · 나머지: 비워도 됨 → "저장 후 계속"
  4. 범위 페이지: 건드리지 않고 "저장 후 계속"
  5. 테스트 사용자 페이지:
    "+ ADD USERS" → 내 구글 이메일 입력 → 추가 → "저장 후 계속"
    ⚠️ 여기서 이메일을 추가하지 않으면 나중에 구글 로그인이 안 됩니다. 반드시 추가하세요.
  6. "대시보드로 돌아가기" 클릭

Client ID & Secret 발급

확장프로그램이 구글 시트에 접근할 수 있는 키를 만듭니다

  1. 사용자 인증 정보 열기 →
  2. 상단 "+ 사용자 인증 정보 만들기" → "OAuth 클라이언트 ID"
  3. 애플리케이션 유형: "웹 애플리케이션"
  4. 이름: 아무거나 (예: Amaranth Check)
  5. "승인된 리디렉션 URI"는 지금 건드리지 않아도 됩니다
    확장프로그램 URI는 6단계에서 등록합니다
  6. "만들기" 클릭 → 팝업에 Client ID와 Client Secret이 표시됩니다
⚠️ 지금 바로 복사하세요. 팝업을 닫으면 Client Secret을 다시 볼 수 없습니다.
Client ID를 입력해주세요
Client Secret을 입력해주세요

Chrome 확장프로그램 설치

Chrome에 플러그인을 올리고 확장프로그램 ID를 확인합니다

  1. Chrome 주소창에 아래를 입력하고 Enter
    chrome://extensions
  2. 화면 우측 상단 "개발자 모드" 토글을 켭니다 (파란색)
  3. "압축해제된 확장 프로그램을 로드합니다" 클릭
    marketing-asset → Amaranth_Check 폴더를 선택하세요
  4. "Amaranth 출퇴근 체크" 카드가 생기면 성공
  5. 카드 하단에 표시된 영문+숫자 32자리 ID를 복사합니다
chrome://extensions 화면에서 카드 하단 ID 문자열
확장프로그램 ID 32자리를 입력해주세요

추가할 Redirect URI:

(이전 단계에서 ID를 입력하면 자동으로 생성됩니다)
  1. 사용자 인증 정보 열기 →

  4. ⚠️ URI 끝에 /가 있는지 확인하세요.
