## 사용 방법

### 클론 (처음 시작)
```bash
git clone https://github.com/likelion-cdc/marketing-asset.git
```

### 글로벌 가이드 업데이트 받기
```bash
git pull origin main
```

### 개인 가이드 관리
글로벌 가이드를 그대로 쓰거나, 개인 브랜치에서 추가 메모를 관리하세요.

```bash
# 개인 브랜치 생성 (최초 1회)
git checkout -b guide/이름

# 개인 메모 추가 후 커밋
git add .
git commit -m "개인 메모 추가"
```

> 글로벌 가이드(`main` 브랜치) 수정이 필요하면 PR을 올려주세요.
