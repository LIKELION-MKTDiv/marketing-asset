# 경쟁사 분석

KDT 부트캠프 경쟁사 모니터링 파이프라인을 실행합니다.

## 실행 순서

1. `cd C:/Users/manid/competitor-analysis` 로 이동
2. `.env` 파일에 API 키가 설정되어 있는지 확인
3. 파이프라인 실행:

```bash
cd /c/Users/manid/competitor-analysis && python main.py
```

## 에이전트 구조

```
오케스트레이터 (orchestrator.py)
├── [1] 조사 에이전트 (researcher.py)      ← 경쟁사 3곳 + 자사 스크래핑
├── [2] 분석 에이전트 (analyzer.py)        ← 경쟁 관계 분류 및 변경사항 정리
├── [3] 인사이트 에이전트 (insight_generator.py) ← 전략 인사이트 도출 (Opus)
└── [4] 문서화 에이전트 (documenter.py)    ← Notion 기록 + Slack 알림
```

## 캐시 동작

- 첫 실행 시 전체 데이터 수집
- 이후 실행 시 `data/cache/` 와 비교해 **변경분만** 분석
- 캐시 초기화: `data/cache/` 폴더 삭제 후 재실행

## 산출물

- `outputs/YYYY-MM-DD_HHMM/research.json` — 원시 조사 데이터
- `outputs/YYYY-MM-DD_HHMM/analysis.json` — 경쟁 분석 결과
- `outputs/YYYY-MM-DD_HHMM/insights.json` — 전략 인사이트
- `outputs/YYYY-MM-DD_HHMM/final.json` — 최종 요약
- Notion 페이지 (자동 생성)
- Slack 알림 (자동 전송)
