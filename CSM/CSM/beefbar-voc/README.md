# 설문 응답 자동 통합·알림

구글폼으로 들어오는 설문 응답을 여러 시트에서 자동으로 합산하고, 목표 응답 건수에 도달하면 Slack으로 알림을 보내는 Apps Script. 응답이 일정 수준(예: 50건) 쌓이면 정성 분석(가설-근거-검증 방식)에 활용합니다.

## 사용법

1. 응답 스프레드시트에서 확장 프로그램 → Apps Script 열기
2. `응답50건_슬랙알림_AppsScript.gs` 내용 붙여넣기
3. `SHEET_A_ID`, `SHEET_B_ID`, `SLACK_WEBHOOK_URL`을 본인 값으로 채워넣기
4. 트리거(양식 제출 시 실행) 등록

## 데이터 보안

- 이 폴더에는 실제 설문 응답 데이터·분석 결과는 포함하지 않습니다. 자동화 코드만 공개합니다.
- 원본 스크립트에 있던 실제 스프레드시트 ID·Slack 웹훅 URL은 플레이스홀더로 교체했습니다.
