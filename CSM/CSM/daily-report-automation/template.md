## 작성 규칙 (템플릿 안내용, 실제 발행 시 삭제)

1. **불릿 스타일**: "이름(번호): 내용" 한 줄로 붙이지 말 것. 이름 불릿 아래 내용을 서브불릿으로 한 번 더 들여쓰기.
2. **아웃콜 시트 링크는 각 섹션 중간에 넣지 말고, 문서 맨 마지막 "📎 아웃콜 시트" 한 곳에만 정리.**
3. **"🔁 심화콜" 분류는 원본 자료에 "심화"라는 표현이 명시적으로 있을 때만 사용.** "고민 중", "재연락 예정", "추후 연락" 같은 표현만으로 임의로 심화콜로 분류하지 말 것 — 이런 경우는 기본적으로 ✅ 착신 완료에 넣고 내용에 후속조치 필요 여부를 서술.
4. **과정명에 기수번호 반드시 포함** (예: BEJV27, GM06, CLD08) — 기수번호는 그날 실제 진행 중인 기수 기준.
5. **번호 체계(1,2,3...)는 그날 실제 진행한 아웃콜 종류에 따라 유동적** — 고정된 섹션 개수가 아니므로 그날 없는 섹션은 건너뛰고 번호를 이어서 매길 것.
6. **"📌 체크 사항" 섹션은 쓰지 않음** — 불필요하다고 판단해 제외(2026-07-08 결정). 특이사항은 본문 각 섹션 안에서 서술.
7. **"완료 X명 / 발신 X명 (수신불가 X명) · 착신율 X%" 통계 줄은 모든 과정 섹션에 예외 없이 넣을 것** (1차/2차뿐 아니라 지원완료독려·합격자대상·결제안내도 동일). 원본 자료에 숫자가 없으면 나열된 이름 수를 세서 계산: 발신=해당 리스트 전체 인원, 완료=수신불가/부재중 제외 인원(취소·거절도 통화 자체는 연결됐으므로 완료로 카운트), 착신율=완료÷발신.
