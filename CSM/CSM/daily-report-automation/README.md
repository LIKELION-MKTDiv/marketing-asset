# 일별 아웃콜 결과 리포트 (반자동화)

부트캠프 아웃콜(1차/2차 콜, 합격자 콜, 결제 안내 콜)을 매일 수행한 뒤, 그 결과를 세 가지 용도로 나눠 정리하는 리포트 양식.

## 템플릿 구성

| 파일 | 용도 |
|---|---|
| `template.md` | 슬랙 공유용 — 과정별 완료/발신/착신율 요약 + `@channel` 공지 문구 |
| `template_전사공유용.md` | 전사 공유용 — 과정별 발신·착신·수신불가·착신율을 표 하나로 압축 |
| `template_내부운영용.md` | 내부 운영용 — 고민 사유, 현장 피드백, 후속 조치가 필요한 대상까지 상세 기록 |

## 개인정보 마스킹 규칙

내부운영용 템플릿에서 개인 단위로 기록할 때는 `성*명(0000)` 형식(이름 가운데 마스킹 + 연락처 끝 4자리)을 사용합니다. 동명이인 구분은 가능하되 전체 PII는 노출하지 않습니다.

## 작업 흐름

1. 그날 아웃콜 수행 후 시트에 결과 입력
2. 위 세 템플릿 중 공유 대상에 맞는 것을 골라 그날 날짜로 리포트 작성
3. 슬랙 공유용은 `@channel` 공지로, 내부운영용은 후속 조치 트래킹용으로 사용
