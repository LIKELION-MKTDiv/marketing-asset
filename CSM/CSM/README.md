# CSM

CDChapter CSM(고객 성공 매니저) 업무에서 만든 자동화·자료 모음. 부트캠프 아웃콜 운영, VOC(고객의 목소리) 분석, 팀 업무 표준화 관련 산출물을 정리합니다.

## 폴더

| 폴더 | 내용 | 상태 |
|---|---|---|
| [`voc-report-automation/`](./voc-report-automation) | 채널톡 VOC 위클리 리포트 자동 수집·집계·발송 파이프라인 | ✅ 공개 |
| [`outcall-training/`](./outcall-training) | 아웃콜 파견인력 교육자료(롤플레잉 시뮬레이션) | ✅ 공개 |
| [`daily-report-automation/`](./daily-report-automation) | 일별 아웃콜 결과 리포트 반자동화 | ✅ 공개 |
| [`channeltalk-rules/`](./channeltalk-rules) | 채널톡 상담 태그·ALF 프롬프트 규칙 정리 | ✅ 공개 |
| [`beefbar-voc/`](./beefbar-voc) | 설문 응답 자동 통합·알림 (클라이언트 실데이터 제외) | ✅ 공개 |
| `report-examples/` | 상사 보고용 분석 리포트 예시 | 🔜 정리 예정 |

각 폴더는 준비되는 대로 하나씩 채워 넣을 예정입니다.

## 데이터 보안 원칙

- 실제 고객·수강생 개인정보(이름, 연락처, 상담 원문 등)가 담긴 원본 데이터는 이 저장소에 올리지 않습니다.
- 올라가는 건 코드, 방법론 문서, 익명화된 집계 수치, 마스킹 처리된 예시뿐입니다.
