### 캠프파이어_소개 ✅ (신규)
캠프파이어는 수업(화상)과 소통(채팅)을 한곳에서 하는 과정 공식 공간이에요. 별도 회원가입 없이 멋쟁이사자처럼 홈페이지 계정으로 입장하실 수 있습니다. 멋사 홈페이지 로그인 → 내 강의실 > 강의목록 → 듣고 계신 과정 카드에서 [캠프파이어 입장하기] 버튼을 클릭하시면 자동으로 입장됩니다.

### 캠프파이어_입장안됨 ✅ (신규)
입장 버튼이 안 보이시거나 입장이 안 되시는 경우, 아직 과정 등록이 완료되지 않았을 가능성이 있습니다. 담당 매니저님께 문의 주시면 확인 도와드리겠습니다.

### 캠프파이어_비밀번호 ✅ (신규)
캠프파이어 최초 입장 시 비밀번호를 설정하시게 되며, 이 비밀번호로 이후 데스크톱 앱과 PC에서 로그인하실 수 있습니다(모바일 앱은 출시 예정). 설정하신 비밀번호는 꼭 기억해 주시기 바랍니다.

### 캠프파이어_모바일지원 ✅ (신규)
캠프파이어 모바일 앱(iOS·Android)은 현재 출시 예정입니다. 출시 이후에는 최초 입장 시 설정하신 비밀번호로 앱에서도 로그인하실 수 있습니다. 현재는 PC(Chrome 브라우저 권장)로 참여하시는 것을 권장드립니다.

### 캠프파이어_권장브라우저 ✅ (신규 — 라이브 가이드 반영)
캠프파이어의 화상 수업·화면 공유·녹화 기능은 **PC Chrome 최신 버전**에 최적화되어 있습니다. Safari·Edge 또는 카카오톡·슬랙 등 인앱 브라우저로 링크를 여시면 카메라·마이크·화면 공유가 제한될 수 있으니, 가능하면 PC에서 Chrome으로 접속해 주시기 바랍니다. 화상·소리·입장 문제 대부분은 이 브라우저 설정 때문인 경우가 많아, 다른 해결 방법을 시도하기 전에 먼저 확인해 주세요.

### 캠프파이어_소리안들림 ✅ (신규)
① 먼저 PC Chrome 최신 버전으로 접속하셨는지 확인 부탁드립니다. ② PC·브라우저 볼륨이 음소거 상태는 아닌지 확인 부탁드립니다. ③ 강의 화면에서 다른 분이 말씀 중이신지(마이크 표시) 확인 부탁드립니다. ④ 그래도 안 되시면 강의에서 한 번 나갔다가 다시 들어와 보시는 것을 권장드립니다.

### 캠프파이어_마이크안됨 ✅ (신규)
① 먼저 PC Chrome 최신 버전으로 접속하셨는지 확인 부탁드립니다. ② Chrome 주소창 왼쪽 자물쇠 아이콘 > 사이트 권한에서 마이크가 [허용]으로 설정되어 있는지 확인 부탁드립니다. ③ 강의 하단 마이크 버튼 옆 화살표에서 올바른 마이크가 선택되어 있는지 확인 부탁드립니다. ④ Zoom 등 마이크를 사용하는 다른 프로그램이 실행 중이라면 종료 후 다시 시도 부탁드립니다.

### 캠프파이어_카메라안됨 ✅ (신규 — 라이브 가이드 반영, 기존 누락)
① 먼저 PC Chrome 최신 버전으로 접속하셨는지 확인 부탁드립니다. ② Chrome 주소창 왼쪽 자물쇠 아이콘 > 사이트 권한에서 카메라가 [허용]으로 설정되어 있는지 확인 부탁드립니다. ③ 강의 화면 하단 카메라 버튼 옆 화살표에서 올바른 카메라 기기가 선택되어 있는지 확인 부탁드립니다. ④ 다른 프로그램(Zoom 등)이 카메라를 사용 중이라면 종료 후 다시 시도해 주세요. 참고로 강의 입장 시 카메라는 기본적으로 켜진 상태로 시작됩니다.

### 캠프파이어_자꾸튕김 ✅ (신규)
① 먼저 PC Chrome 최신 버전으로 접속하셨는지 확인 부탁드립니다. ② 인터넷 연결 상태를 확인해 주시고, 가능하시면 유선 인터넷 사용을 권장드립니다. ③ 내 강의실 > 캠프파이어 입장하기로 다시 들어오시면 됩니다(로그인은 유지되어 다시 로그인하실 필요는 없습니다). ④ 브라우저 새로고침 또는 데스크톱 앱 사용도 도움이 될 수 있습니다.

### 캠프파이어_1대1통화 ✅ (신규 — 라이브 가이드 반영, 기존 누락)
캠프파이어에서 음성·화상 통화는 **채널이 아닌 DM(다이렉트 메시지, 1:1 대화)에서만** 가능합니다. 대화 상대와의 DM 화면 상단, 이름 옆 전화·화상 아이콘을 누르시면 상대에게 통화 요청이 가고, 상대가 수락하면 통화가 연결됩니다. 통화 종료 후에는 DM에 통화 시작/종료 기록이 남고, 상대가 받지 못하면 '부재중 통화'로 표시됩니다.

### 캠프파이어_알림설정 ✅ (신규 — 라이브 가이드 반영)
캠프파이어 알림 기본값은 **'멘션만'**(나를 @멘션할 때만 알림)로 설정되어 있습니다. 왼쪽 아래 프로필 > 설정 > 알림 탭에서 전체 메시지 알림으로 변경하시거나 알림을 끄실 수 있고, 채널별로도(채널에 마우스를 올려 나오는 케밥(⋮) 메뉴에서) 따로 설정 가능합니다. 다만 일정·과제 안내 등 중요한 공지를 놓치지 않으시려면 **공지 채널만은 '전체 메시지'로 켜두시는 것**을 권장드립니다.

### 캠프파이어_녹화영상 ✅ (신규)
수업 녹화 영상은 강사님이 녹화 설정을 켜신 경우에 한해 제공되며, 확인 방법은 과정마다 다를 수 있습니다(공유 링크, 별도 안내 등). 담당 매니저님·강사님 안내를 참고해 주시기 바랍니다.

### 캠프파이어_출석기준 ✅ (신규)
캠프파이어 수업 출석은 수업 시간 중 약 10분마다 카메라 화면이 자동 캡처되어 기록됩니다. 카메라가 꺼져 있으면 '스킵'으로 남아 출석 누락 처리될 수 있으니, 과정 규칙에 따라 카메라를 켜 주시기 바랍니다.

### 캠프파이어_닉네임변경 ✅ (신규)
프로필에서 표시 이름을 변경하실 수 있습니다. 다만 출석 확인과 원활한 소통을 위해 되도록 본명을 유지해 주실 것을 권장드립니다.

캠프파이어 사용법 전반(입장, 채팅, 화상수업 참여, 문제해결 등)은 아래 가이드 문서에서 확인하실 수 있습니다.
