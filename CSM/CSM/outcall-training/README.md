# 아웃콜 스크립트·롤플레잉 시뮬레이터

부트캠프 아웃콜 파견인력이 실전 콜 전에 연습할 수 있도록 만든 브라우저 기반 도구. 상황별 스크립트와 문자 발송 양식을 탭으로 모아두고, 실제 상담처럼 대사를 주고받는 롤플레잉 시뮬레이션 기능을 포함합니다.

## 사용법

`롤플레잉_시뮬레이터.html` 파일을 브라우저로 열면 바로 실행됩니다(별도 설치·서버 불필요).

## 데이터 보안

이 파일에 담긴 이름·연락처·이메일은 전부 `{지원자명}`, `{과정명}`, `{이메일주소}` 같은 **틀(placeholder)** 이며, 실제 고객·수강생 정보는 포함되어 있지 않습니다.
